# 邮件群发系统

基于 PHP + SQLite + PHPMailer 的批量邮件群发系统，支持多 SMTP 账号轮询、邮件模板、收件人分组、发送日志等功能。

## 功能特性

- **多 SMTP 账号轮询**：自动分配发送任务到多个账号，支持日发送限额
- **邮件模板**：预定义邮件内容，快速选择发送
- **收件人分组**：分组管理收件人，支持批量导入
- **黑名单过滤**：自动过滤黑名单邮箱
- **发送日志**：完整记录每次发送结果，支持统计和筛选
- **分批发送**：支持设置每批数量和发送间隔
- **重试机制**：发送失败自动重试

## 安装

1. 确保 PHP >= 7.4 并开启 sqlite3、openssl 扩展
2. 将代码上传到网站目录
3. 访问 `/install/` 完成安装
4. 安装后删除 install 目录或确保其不可访问

## 使用

1. 登录管理后台 `/admin/`
2. 添加 SMTP 发送账号
3. 创建邮件模板（可选）
4. 添加收件人或收件人分组
5. 访问前台 `/index.html` 发送邮件

## 目录结构

```
├── admin/          # 管理后台
├── api/            # API 接口
├── data/           # SQLite 数据库
├── install/        # 安装程序
├── vendor/         # PHPMailer 库
├── index.html      # 前台发送页面
└── zhuan.html      # 邮箱格式化工具
```

## 技术栈

- PHP 7.4+
- SQLite3
- PHPMailer 6.x
- 原生 HTML/CSS/JS
